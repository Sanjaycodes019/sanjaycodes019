**CargoNepal, Full-Stack Logistics Platform**
MERN-based logistics platform with role-based dashboards across 3 user types (MVC + OOP architecture).
- 15+ RESTful API endpoints with JWT authentication and OTP verification
- Real-time conflict resolution via Socket.io to eliminate double-booking
- OpenCage Geocoding API integration for live truck discovery, deployed to production on Vercel
- Live Demo: https://cargo-nepal-final.vercel.app/
- GitHub: https://github.com/Sanjaycodes019/cargonepal_final

**SafeHer, Blockchain Safety Companion**
A decentralized women's safety application combining AI, IPFS, and Ethereum blockchain for tamper-proof emergency response.
- One-tap SOS alerts with real-time location sharing to emergency contacts
- Evidence stored on IPFS and incidents recorded on the Ethereum blockchain for permanent verification
- Emergency guidance chatbot powered by a locally hosted AI model (Ollama) via FastAPI backend
- Built as Team Lead for a 4-member team; Top 15 finish among 100+ teams at HackIndia Spark 5

Desktop application for simulating and visualizing CPU scheduling algorithms with console and graphical interfaces.
